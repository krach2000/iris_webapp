Whenever a user changes an input (i.e. either the sepal / petal type or the number-of-clusters slider), the k-means Python function is updated and the visuals (i.e. the scatter plot showing the clusters and their centroids) changes automatically. This demonstrates that we can develop seemingly simple interactive webapps that are powered by advanced analytics under the bonnet.
